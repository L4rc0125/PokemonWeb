PokemonWeb の使い方

## 起動方法

1. Python を入れる
2. このフォルダに移動する

   ```powershell
   cd C:\Users\81809\Desktop\PokemonWeb
   ```

3. 必要なパッケージを入れる

   ```powershell
   pip install flask requests
   ```

4. アプリを起動する

   ```powershell
   python app.py
   ```

5. ブラウザで以下を開く

   http://127.0.0.1:5000

## 現在できること

- PokeAPI から取得したポケモンデータをキャッシュして使う
- 最終進化ポケモンを中心に検索する
- ポケモンのタイプ、特性、技、ステータスを表示する
- 攻撃側と防御側を並べてダメージ計算用の条件を選ぶ
- フォーム違い、メガシンカ、リージョンフォームなど一部の特殊フォームを扱う
- ブラウザ側で一部のダメージ計算ロジックを動かす
- ローカルストレージにポケモンスロットや努力値スライダーの状態を保存する

## 注意点

- `/calculate` API はまだ本実装ではなく、現在は `ダメ計未移植` を返すだけです。
- ダメージ計算の多くは `templates/index.html` の JavaScript 側にあります。
- キャッシュがない初回起動時は、PokeAPI からデータを取得するため時間がかかります。
- ネットワークに接続できない場合、未キャッシュのポケモンや技は取得できないことがあります。

## 次にやるとよいこと

- `app.py` の `/calculate` にダメージ計算ロジックを移植して API 化する
- `templates/index.html` を HTML / CSS / JavaScript に分割する
- 特殊フォーム、専用技、特性上書きを JSON ファイルへ移して編集しやすくする
- 起動時やキャッシュ作成中のエラー表示を分かりやすくする
- OCR を使うなら、画面キャプチャや補助アプリとの連携方法を整理する
